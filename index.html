04

Blog Connection

01

조형물 / 경관

대표 프로젝트 이미지를 중심으로 회사가 수행한 주요 분야를 직관적으로 소개합니다.

02

3D 모델

완성된 결과물뿐 아니라 모델 기반의 구조 이해까지 가능하도록 확장합니다.

03

GS 공간 체험

가우시안 스플래팅 데모를 통해 실제 공간을 브라우저 안에서 체험하게 합니다.

04

브랜드 확장

블로그 등 외부 채널과 연계해 회사의 레퍼런스와 콘텐츠를 더 깊게 탐색하게 합니다.

3D PROJECT VIEWER

실제 GLB 모델 뷰어

업로드한 GLB 파일을 활용해 브라우저 안에서 직접 회전·확대 가능한 3D Viewer 영역을 구성했습니다.

GLB Model Camera Controls Auto Rotate
마우스로 드래그하면 모델을 회전할 수 있고, 휠로 확대/축소할 수 있습니다.

적용 포인트

  • 회사 기술력을 단순 이미지가 아닌 모델 기반 소개로 확장
  • 조형물의 입체 구조와 비례를 이해하기 쉬운 방식으로 표현
  • 프로젝트별 GLB 파일로 교체 가능한 공통 3D 영역 구성
  • 보고용 데모에서 실제 홈페이지 구현 방향까지 설명 가능
